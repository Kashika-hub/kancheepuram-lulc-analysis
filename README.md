# Land Use/Land Cover Change Detection using Google Earth Engine

## Overview

This project focuses on Land Use/Land Cover (LULC) classification and temporal change detection for Kancheepuram district using Sentinel-2 and Landsat imagery in Google Earth Engine. The analysis was performed to study urban expansion, vegetation dynamics, water body variation, and barren land changes between 2015 and 2025.

---

## Objectives

* Perform supervised classification
* Analyze land use changes from 2015 to 2025
* Generate thematic maps
* Study urban expansion and vegetation changes

---

## Dataset Used

* Landsat Imagery
* Google Earth Engine Datasets

# Key Observations

* Vegetation area increased significantly between 2015 and 2025.
* Water body coverage showed moderate improvement over the study period.
* Barren land area decreased over time.
* Spatial analysis highlights noticeable land cover transformation across Kancheepuram district.
